# Metronome

A small retro metronome app with reliable playback controls, adjustable tempo, and selectable time signatures.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Notes

- Built with Vite and React.
- Audio feedback uses the bundled click samples in `src/sounds/`.
- The UI keeps the original retro character, but the control flow and layout are updated for accessibility and smaller screens.
