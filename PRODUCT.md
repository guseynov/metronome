# Product

## Register

product

## Users

Musicians, students, and practicing players who need a fast way to set tempo, switch meter, and keep time while rehearsing.

## Product Purpose

Provide a reliable metronome with immediate tempo control, clear beat feedback, and low-friction playback controls so the user can stay focused on practice instead of the interface.

## Brand Personality

Calm, precise, dependable.

The interface should feel like a trustworthy instrument: clear, familiar, and lightly tactile, without gimmicks or visual noise.

## Anti-references

- Flashy music toys that optimize for novelty over control.
- Over-decorated skeuomorphic interfaces.
- Marketing-style product chrome that makes a tool feel like a landing page.
- Novel custom controls that force the user to relearn standard interactions.
- Heavy decorative motion that competes with the task.

## Design Principles

- Familiar controls first.
- State changes should be obvious at a glance.
- Keep the layout dense enough for fast use, but never cramped.
- Use color sparingly for selection and feedback, not decoration.
- Make keyboard and small-screen use feel first-class.

## Accessibility & Inclusion

- Target WCAG AA contrast.
- Keep all controls keyboard accessible.
- Preserve reduced-motion support.
- Avoid relying on color alone to communicate state.
- Ensure the BPM readout, labels, and meter controls remain legible on mobile screens.
